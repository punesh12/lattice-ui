# @punesh12/lattice-ui

React component library for [Lattice UI](https://github.com/punesh12/lattice-ui).

## Install

```bash
pnpm add @punesh12/lattice-ui @punesh12/lattice-ui-tokens
```

Requires **React 18 or 19**. See the [installation guide](https://github.com/punesh12/lattice-ui#installation) in the main repository.
